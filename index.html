Multiscript is a cross-platform desktop application for combining Bible versions for multicultural Christian ministry. It makes it easy to collate Bible passages and versions in all the combinations you need.

Why Multiscript?

As our world becomes more globalised, more Christian ministries are becoming multicultural. These ministries regularly need copies of the Bible with multiple versions shown in parallel. For example, a Bible study may have some guests who need an English and Chinese version together, and other guests who need an English and Spanish version. The more multicultural the ministry, the larger the number of combinations required.

One solution to this problem is to have parallel printed Bibles for everyone. But many ministries don’t have access to a large library with enough Bibles for every guest and every language combination. A common alternative is to access Bible websites that offer multiple translations in a parallel layout. Yet there remain times when a printed handout is preferable – especially if you want to see more text than can fit on a screen, or encourage guests to write notes directly on the page. You might have tried manually  to create handouts of the exact passages and versions that you need. But as the number of languages increases, the number of combinations quickly becomes too large to generate manually.

Multiscript addresses this problem. You provide the app with a list of the Bible references and versions you need, as a well as a template for the handout you’re producing. Multiscript collects the passages online, and uses the template to create a separate file for each version combination. These files can then be printed, subject to copyright and licensing restrictions.
